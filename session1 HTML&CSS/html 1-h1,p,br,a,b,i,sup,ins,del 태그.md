p - paragraph, 문단을 구성한다.
br - break line, 줄바꿈을 실행한다
	
    <p>본문을 구성하고 <br /> 줄바꿈을 실행</p>


<문자열 태그>
b - bold 처리
i - italic 처리
del - 지움줄 처리
sup - 거듭제곱 처리
ins - 밑줄 처리
a - anchor 위치이동을 처리한다.

<앵커 태그의 속성>
href - 이동할 위치의 주소값
target - _blank: 새 창을 띄움, _self:현재 창에서 띄움

    <b>Seoul</b>
    <i>Seoul</i>
    <del>Seoul</del>
    Seoul<sup>Seoul</sup>
    <ins>Seoul</ins>
    <a href="www.google.com" target="_blank">구글</a>
    <a href="www.naver.com" target="_self">네이버</a>
